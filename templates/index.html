Collaboration is important in this class and beyond. I'll list the most important things I've learned so far.

  1. In groups, tasks should always be delegated so it's clear to all members who's responsible for what
  2. Set checkpoints for the delegated tasks, and lean on the rigid side. People respond well to a challenge.
  3. This one's a time saver. If a group member is stuck on a problem, refer them to a guide if you remember the solution showing up there.
  4. Actually trust the group members to do their part! (in other words don't stress yourself by doing everything.
  5. Don't be afraid to ask other group members or teachers for help if needed!
